
# Predictive Pulse: Harnessing Machine Learning for Blood Pressure Analysis

## Project Overview
Predictive Pulse is a machine learning web application that predicts hypertension using health data.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript

## How to Run

1. Install libraries

pip install flask pandas numpy scikit-learn joblib

2. Run project

python app.py

3. Open browser

http://127.0.0.1:5000
